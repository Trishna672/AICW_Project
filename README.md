
This project uses a Random Forest Classifier, a supervised machine learning algorithm, to classify SMS messages as either spam/phishing or legitimate. The model is trained on a labeled SMS dataset and learns patterns commonly found in fraudulent messages. Before training, messages are converted into numerical features using TF-IDF vectorization, which captures the importance of words in each message. The trained model analyzes incoming SMS content and predicts the likelihood of it being spam. It achieves high accuracy in detecting suspicious messages and provides a probability score to support its prediction. The model is saved using Pickle and integrated into a Streamlit application for real-time phishing detection.
