# AICW_Project
This project is a Machine Learning–based SMS phishing and spam detection system that identifies whether a message is legitimate or potentially fraudulent. It uses Natural Language Processing (NLP) techniques to preprocess and analyze SMS content. Messages are cleaned, transformed using TF-IDF vectorization, and classified using a Random Forest Classifier. The model predicts the probability of a message being spam or safe. In addition to machine learning predictions, the system performs rule-based analysis to detect common phishing patterns such as urgent requests, fake rewards, and account verification scams. The application provides a risk level (Low, Medium, High, or Very High) along with a detailed message analysis. A user-friendly Streamlit interface allows real-time message testing. The project demonstrates the practical application of NLP and Machine Learning in cybersecurity. It aims to help users identify and avoid phishing attacks through intelligent message analysis.
